O Poder do Flexbox

Flexbox é um modo de layout unidimensional para organizar itens em linhas ou colunas. Ele é perfeito para alinhar elementos de forma equilibrada em qualquer tela.

Entendendo o Box Model

O Box Model define como margens, bordas e preenchimentos interagem para determinar o tamanho total de um elemento na tela.
